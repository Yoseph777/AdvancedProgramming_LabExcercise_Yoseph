**Lab Exercises Portfolio: Advanced Programming**
This repository contains a series of 4 comprehensive lab exercises designed to demonstrate core concepts in full-stack development, desktop GUI frameworks, state management, and real-time networking.
 Lab Exercises Overview
 Lab 1: Poker Game
A desktop-based multiplayer poker simulator featuring custom UI components, automated card dealing, and dynamic hand evaluation.
* Key Features:
    * Simultaneous 3-player layout with custom CSS cards formatting.
    * Dynamic poker hand evaluation (Straight, Flush, Full House, etc.).
* Tech Stack: Java, JavaFX, CSS.
 Lab 2: Chat Application
A lightweight, multi-threaded networking application enabling real-time text communication between clients via a local server.
* Key Features:
    * Multi-threaded server architecture to handle concurrent connections.
    * Graceful client connection/disconnection handling.
    * Instant message broadcasting and custom user handles.
* Tech Stack: Java (Sockets, ServerSocket), Threading, Embedded Database (MySQL).
 Lab 3: Notepad
A clean, functional text-editing application focused on smooth file I/O operations and local storage management.
* Key Features:
    * Standard text manipulation tools (Create, Read, Update, Delete text files).
    * Native OS file explorer integration for saving and opening directories.
    * Can read PDF Documents.
* Tech Stack: JavaFX / Frontend UI, Java File I/O API.
 Lab 4: Integrated Registry & Communication Hub
This system manages Academic Registries while facilitating direct communication between Teacher and Student.
* Key Features:
    * Role-Based Access Control (RBAC): Distinct secure login portals for Teachers and Students.
    * Academic Registry: Full CRUD operations to manage student enrollment, grades, and teacher profiles.
    * Contextual Chat Environment: Integrated chat rooms that initialize instantly post-login to bridge teacher-student interaction.
* Tech Stack: Java, Embedded Database (MySQL), WebSockets.
